    About

    At Samyukti, we wanted to create something small, meaningful for us to teach, learn and practice Ruby on Rails. We had then come up with a simple Library Management web application to explore Ruby on Rails. As the project progressed, it seemed a right decision to make it an open source software for the benefit of larger community. We intend to maintain this as a stable software in the foreseeable future.

    Features

    A few things that come into our mind when we hear 'Library' are books, authors, copies, memberships, members etc. We chose to call books and authors as items and creators respectively, to make it a little generic and go beyond the scope of just books.

    The features include,

    • Manage Items with Creators and Copies
    • Manage Memberships and Members
    • Issue an item
    • Reserve an item
    • Record Receipts towards membership fees, fines and contributions
    • Dashboard and reports

    We have chosen to keep simple relationships between the entities for keeping the design and application simple. Thus, one issue transaction can have only one copy. However, to make the issue process faster, we allow cloning issues, reservations, books, copies and all such things.

    Other features include,

    • Mobile ready, using Bootstrap framework
    • User authentication and authorization
    • A public page
    • Member access (under development)
    • Settings (under development)

    Technical Details

    Samgrah is built with Ruby on Rails (4.1.x) and PostgreSQL.

    Please refer the licenses directory for different components used in this project.

    Installation

    The steps to install the software is given in the README of the repository. At the moment, we recommend installing in one of the Linux variants for production use.
